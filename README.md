# 🎬 React Movie App

A modern and responsive movie browsing application built with **React.js**.  
This app allows users to search for movies, view details, and explore trending films using a movie API.

---

## 🚀 Features

- 🔍 Search movies by title  
- 🎥 View movie details (poster, rating, description)  
- 🌐 Fetch real-time movie data from API  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- ⚡ Fast performance using React components  
- 🎨 Clean and modern UI  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3  
- REST API (e.g. TMDB API)
